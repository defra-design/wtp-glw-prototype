Unique transaction ID

All waste movements must be accompanied by a Unique Transaction ID (UTID). Parties in the movement must record the UTID and use it to report information to the service.

  • Generate a UTID if you are initiating a waste movement
  • Enter the provided UTID if you are submitting data for a waste movement that's been initiated by another party

Check with your waste producer if you are not sure whether a UTID has been generated for the movement.
