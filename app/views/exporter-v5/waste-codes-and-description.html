Help with identification codes

This could be a:

  • Basel Annex XI code starting with the letter B. For example, B3011
  • OECD code starting with the letter G. For example, GC010 (these codes are only relevant if you do not have a Basel Annex XI code)
  • Annex IIIA code which are for specific mixtures of waste listed in Annex III
  • Annex IIIB code which are for additional green list waste awaiting inclusion into relevant regulation annexes
{# EC list #}
This is also known as an EWC. For example, 19 12 01.
{# National code #}
This is also known as a commodity code and is not required by UK regulators.
{# Description #}
What to include in your description

You should include things like international specification, grade or code. For example, it's:

  • 'EN 643 2.05.00' for paper
  • 'B' for glass
  • 'ISRI 200' for metal

For plastic waste you should include the polymer type. For example, 'LPDE, film' or 'HDPE, tubs'.
